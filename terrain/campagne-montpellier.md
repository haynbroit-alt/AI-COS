# Campagne cabinets recrutement tech — Montpellier/Occitanie (2026-07-21)

Positionnement : **radar de signaux faibles commerciaux** — jamais « une liste
de prospects ». Objectif : un échange de 15 min, puis premier client pilote.
Offre test : 10 opportunités gratuites → RDV généré = paiement au résultat,
sinon aucun engagement → abonnement Radar 199 €/mois après validation.

Livrable : `velyx-rapport-montpellier.md` (6 opportunités, format
Entreprise/Signal/Priorité/Décideur/Angle/Pourquoi maintenant).

Envois automatiques via la boucle (template `cabinet`, cap 5/jour, relance
unique J+2, opt-out). Priorité de file : HOT → pool existant → WARM → ESN.
